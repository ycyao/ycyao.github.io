#01-二分查找
问题描述
二分法是这样定义的：
给定n个从小到大排序好的整数序列List[],以及待查找的整数x,我们的目标是找到x在List中的下标。否则，
返回-1表示没找到。
二分法是先找到序列的中点List[M],与x进行比较，相等返回中点下标；否则，若List[M]>x,则在左边的子系列
中查找x;若List[M]<x，则在右边的子系列中查找x。

函数代码如下：<–!more–>
Position BinarySearch( List L, ElementType X )
{
//struct LNode left,right,mid;
int left=1,right=L->Last;
while(left<=right)
{
int mid=left+((right-left)>>1);
if(L->Data[mid]>X)
{
right=mid-1;
}
else if(L->Data[mid]<X)
{
left=mid+1;
}
else
{
return mid;
}
}
return NotFound;
}
