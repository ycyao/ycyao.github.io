169求众数

问题描述
给定一个大小为 n 的数组，找到其中的众数。众数是指在数组中出现次数大于 ⌊ n/2 ⌋ 的元素。
你可以假设数组是非空的，并且数组中的众数永远存在。

摩尔投票算法

介绍:

是一种在线时间O(n)和空间复杂度的情况下，在一个元素序列中查找包含最多的元素，是一种典型的流算法。
在它最简单的形式是查找最多的元素，即超过一半以上的元素。如果系列中没有最多的元素，算法不能检测正确结果，只是将输出其中的一个元素之一。
当元素重复次数比较小时，对于流算法不能在小于线性空间的情况下查找频率最高的元素。

<–!–more–>

描述：

1.定义一个result和count(结果和计数器),计数器初始化为0；
2.依次遍历序列中的元素，当处理元素x时：
     如果计数器为0，将x赋给result，计数器置为1
     如果计数器不为0，比较result和x，相等计数器加1，不相等计数器减1
3.循环结束，result既为最多的元素

理解：

对于基本的摩尔投票问题，找出一组数字序列中出现次数大于一半的数字。显然问题中数字只有一个，如果存在的话。
摩尔投票算法也是基于这样一个事实：**每次从序列中选择两个不相同的数字相互抵消，最后剩下的一个数字或者几个相同的数字，就是出现次数大于总数一半的那个数字**。
在理解这个事实的基础上去考虑这个算法，就容易多了。

代码如下：

class Solution {
public:
    int majorityElement(vector<int>& nums) {

        int re,count=0;
        for(int va:nums)
        {
            if(count==0)
            {
                re=va;
                count=1;
            }
            else if(re==va)
                count++;
            else count--;

        }
        return re;
    }
};
